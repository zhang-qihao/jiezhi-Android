# 捷职招聘安卓端

## 一、项目技术

源码地址: 

[git@github.com:zhang-qihao/jiezhi-Android.git](git@github.com:zhang-qihao/jiezhi-Android.git)

[git@github.com:zhang-qihao/jiezhi-GoWeb.git](git@github.com:zhang-qihao/jiezhi-GoWeb.git)

###  1、项目语言

* Android
* Golang
* Mysql

### 2、项目框架

**Android:**

* OKHttp	*http 请求框架*
* gson		*json 类型数据解析框架*

**GoWeb:**

* gin		*web 服务框架*
* gorm	*数据库框架*
